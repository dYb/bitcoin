## usage
```
yarn 
npm start
```
http://localhost:3100/page/page1
http://localhost:3100/page/page2
新增页面要增加到`package.json`中的`pages`字段

* /accept/     接受比特币
* /send/       接受比特币
* /extract/    提取比特币
* /info/       个人中心
* /login/      登录
* /password/   设置/修改/重置账号密码
* /wallet/     我的钱包
* /myorder/    我的历史订单
* /buysell/    买卖比特币
* /market/     交易市场
* /ad/         发布广告
